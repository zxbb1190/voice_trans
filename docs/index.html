为游戏语音场景做的实时字幕工具

它不是通用会议软件，而是围绕 PC 游戏、Discord 队友沟通、海外服开黑和直播字幕辅助设计的轻量桌面工具。

🔊

系统声音捕获

通过 WASAPI Loopback 捕获电脑正在播放的声音。优先选择 [系统声音] / Loopback，不需要录你的麦克风。

🧠

本地语音识别

基于 faster-whisper 离线识别语音，Lite 版首次运行下载模型，Full 版内置 Whisper small 模型。

🌐

固定方向翻译

在浮窗标题栏选择识别语言和目标语言，支持英译中、中译英，并尽量保留游戏术语、缩写和点位。

🪟

游戏浮窗

透明置顶字幕窗口，支持拖拽、调整透明度、显示原文和译文，不需要切出游戏查看翻译。

📱

手机同步

同一局域网下用手机浏览器打开页面，即可实时接收字幕，适合把手机当副屏使用。

🛠️

开源可自定义

选择适合你的下载版本

第一次体验建议下载 Lite 版。它体积更小，首次运行会自动从默认源下载 Whisper 模型。网络环境较差时可以选择 Full 版。

推荐

Lite 版

体积更小，适合大多数用户。首次运行时会下载语音识别模型，默认使用 ModelScope 国内源，也可改用 Hugging Face。

适合：第一次尝试、网络可用、想要更小下载包。

四步开始使用

当前仍是早期测试版本。遇到音频设备、识别质量或游戏兼容问题，欢迎到 GitHub 提交 Issue。

下载并解压

下载 Lite 或 Full 包，解压后运行 GameVoiceTranslator.exe。

填写翻译 API

在设置中选择 OpenAI 兼容接口或 Google Cloud Translation，并填写 API Key。

选择系统声音

音频设备优先选择 [系统声音] 或 Loopback。不要选择普通麦克风。

进入游戏测试

播放游戏、Discord 或视频语音，浮窗会显示原文和翻译结果。

常见问题

把用户最担心的问题先说清楚，可以减少误解，也方便你收集更高质量的测试反馈。

它会录我的麦克风吗？

默认目标是捕获 Windows 系统声音输出，不是麦克风。设置里应优先选择 [系统声音] / Loopback 设备；普通麦克风通常录不到游戏声音。

会不会注入游戏进程？

不会。它是独立桌面程序，通过系统声音设备采集音频，并用透明置顶窗口显示字幕，不修改游戏客户端。

为什么没有字幕？

通常是音频设备没选对，或者游戏 / 微信 / Discord 声音走了另一个输出设备。先确认 Windows 音量混合器里的输出设备和工具里选择的 Loopback 设备一致。

为什么识别不准？

真人语音比视频更难识别，尤其是短句、口音、噪声、背景音乐和压缩音频。可以尝试提高模型大小、降低游戏背景音乐、固定识别语言，并反馈样例。

Lite 和 Full 有什么区别？

Lite 包更小，首次运行会下载 Whisper 模型；Full 包更大，内置 Whisper small 模型，适合网络不稳定的用户。

支持哪些游戏？

只要游戏语音能从当前扬声器、耳机、HDMI、USB 声卡或虚拟声卡正常播放，并且系统能暴露对应 Loopback 设备，就可以尝试。未逐个实测的游戏不应标记为已兼容。

帮它变得更适合真实游戏语音

现在最需要的是不同游戏、不同设备、不同语音环境的真实反馈。欢迎提交 Issue：游戏名、音频设备、是否捕获成功、识别是否漏字、延迟体验。
